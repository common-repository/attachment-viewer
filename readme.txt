== Description ==

Attachment Viewer is a widget to place in your blog sidebar to display the latest attachment pictures of your blog.

You can change the widget title, number of items, link destination and displayed category in the widget options.

== Installation ==

1. Upload `attachview.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to `Presentacion->Widget` and add `Latest Picture` widget tyo your sidebar
1. Configure `Latest Picture` widget as you need
